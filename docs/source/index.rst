MoreliaTalk server
******************

**MoreliaTalk server** - used to deploy the MoreliaTalk messenger infrastructure.

The development uses
====================

* `Python 3.10 <https://www.python.org/>`_ - programming language

* `FastAPI <https://fastapi.tiangolo.com>`_ - basic framework

* `SQLObject <http://sqlobject.org>`_ - ORM for working with the database

* `Pydantic <https://pydantic-docs.helpmanual.io>`_ - data validation

* `Starlette <https://www.starlette.io>`_ - lightweight ASGI framework

* `Websockets <https://pypi.org/project/websockets/>`_ - Python implementation of Websockets protocol (RFC 6455 & 7692)
